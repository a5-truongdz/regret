1. register are kinda "immutable"

2. register can hold references to variable, and integers.

keywords:
func <name>,<return type>,<parameter count>,[*parameter(s)]:
    - create a function.
call <function>,[*parameter(s)];
    - call <function>.
def <name>,<data type>,[value];
    - define a variable.
ref <variable>,<register>;
    - create a reference to <variable> inside <register>.
print <register>;
    - print the value of <register>, if <register> is a reference dereference it.

    - print the literal value of <register> (meaning it also print the memory address if holding a reference).
return <value>;

continue;

    - copy STDIN,r1;    // assuming the user input an integers, r1 is None
    - for STDOUT and STDERR, why th do you want to capture it?
add <register 1>,<register 2>,<register 3>
    - <register 3> = <register 1> + <register 2>
    - the same go for sub, div, mul, mod.

example: hello world
